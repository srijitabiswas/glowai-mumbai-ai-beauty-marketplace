# GlowAI Mumbai – AI Beauty & Styling Concierge

GlowAI Mumbai is an AI-powered beauty, grooming, and styling discovery platform designed to help users find the perfect salons, beauty professionals, grooming experts, bridal services, and self-care experiences across Mumbai.

The platform combines AI-powered profile analysis, personalized recommendations, salon discovery, bridal planning, and an intelligent concierge assistant to create a premium and highly personalized experience.

## Installation

### Clone Repository

```bash
git clone https://github.com/srijitabiswas/glowai-mumbai-ai-beauty-marketplace.git
cd glowai-mumbai-ai-beauty-marketplace
```

### Install Frontend Dependencies

```bash
npm install
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Environment Variables

Create:

```bash
backend/.env
```

Add:

```env
OPENAI_API_KEY=your_openai_key
GOOGLE_PLACES_API_KEY=your_google_places_key
GOOGLE_MAPS_API_KEY=your_google_maps_key
PORT=5000
```

---

## Run Locally

### Frontend

```bash
npm run dev
```

### Backend

```bash
cd backend
npm run dev
```
